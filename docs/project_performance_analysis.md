## 複数日保有バックテスト（analyze_multiday.py）

### 保有日数が長いほど成績が悪化（全体）
| 保有期間 | プラス率 | 平均リターン |
|---|---|---|
| 1日後（現行） | 42.7% | +0.13% |
| 3日後 | 33.3% | -1.07% |
| 5日後 | 31.9% | -1.99% |
→ **NORMAL・WEAK日は翌日引け売りが最適**

### 最大リターンの到達日（BUY判定）
- 1日目に最高値: 47%（22件）← 最多
- → 寄り付き直後の指値売りが最も効率的

### 例外：PANIC日は3〜4日保有が有利
| 地合い | 1日後 | 3日後 | 4日後 |
|---|---|---|---|
| NORMAL | +0.1% | -2.3% | -2.2% |
| WEAK | +1.1% | -1.9% | -1.8% |
| **PANIC** | **-0.8%** | **+2.7%** | **+3.4%** |

### PANIC日 × 3日保有シミュレーション（+3%利確/損切り-5%）
- 勝率: **67.5%**、平均: **+0.65%**、損切り発動: わずか2件
- スコア9.5以上のPANIC日銘柄: 3日後avg **+8.1%**
- CAUTION判定のPANIC日銘柄: 3日後avg **+13.3%**（アスタリスク+29%、アミタ+24%など）

### PANIC日の戦略案
1. PANIC翌朝 寄り付きで買い
2. スコア9.0以上を優先
3. BUY/CAUTION両方対象
4. 損切り -5%、3日後引けで決済
- ※ サンプル2日分のみ（3/19・3/23）。1ヶ月後に再評価必須。

## scan_morning PANIC予測の外れ事例（2026-03-27）

### 経緯
- scan_morning: スコア0点 → PANIC予測（CME先物-2.08%、米3指数全落）
- 実際の市場: 騰落比65.9%、日経+0.48% → NORMAL

### 外れた原因（3つ）
1. **CME先物と東京市場の乖離**: 朝8:30時点の先物-2.08%が東京開場前に急反転
2. **前日リバウンド未考慮**: WEAK翌日は売られすぎ反発が起きやすいがスコアリングに未反映
   - 3/12 WEAK→3/13 NORMAL(61.3%)、3/17 WEAK→3/18 NORMAL(75%)、3/26 WEAK→3/27 NORMAL(65.9%)
3. **ドル円159円台の追い風を過小評価**: 現在最大2点だが輸出株への下支え効果が大きい

### PANIC予測精度（4回中3回的中 = 75%）
- 3/3、3/19、3/23: PANIC予測 → 実際PANIC ✅
- 3/27: PANIC予測 → 実際NORMAL ❌（トランプ関税前の特殊相場環境）

### 改善案（1ヶ月後に再評価）
1. 前日WEAK/PANIC時に+1点ボーナス追加
2. CME先物の急落ペナルティを緩和（-1点→0点）
3. ドル円155円超の場合+3点に引き上げ
- ※ 現時点ではデータ不足のため実装保留
